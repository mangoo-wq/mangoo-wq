# SOP Starter Pack (Free)

A lightweight starter pack to run recurring work with less friction.

## Included
- Daily Ops Checklist
- SOP Template
- Next-3 Planning Sheet
- Commit Scope Rules

## Who it's for
- Solo builders/operators
- People who repeat similar tasks daily
- Teams that need predictable execution

## Usage
1. Copy files from `templates/`
2. Fill your workflow details
3. Run for 7 days
4. Keep only what actually helps
