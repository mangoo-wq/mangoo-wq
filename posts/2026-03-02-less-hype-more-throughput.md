# Less Hype, More Throughput

I stopped trying to automate everything.
I now automate only what repeats, hurts, and can be measured.

This week, I switched my daily AI briefing to a free RSS-first pipeline.
No paid APIs, no heavy orchestration. Just cron + Python + clear output.

## What changed
- Daily 08:00 brief runs automatically
- Output is fixed: 3 core lines + 1 NASDAQ impact line
- Sources include official blogs + Hacker News + GeekNews

## Why this matters
The goal is not “more automation.”
The goal is faster, calmer decisions every morning.

## Rule I’m using
If a workflow does not create one concrete action, it gets rewritten or removed.
